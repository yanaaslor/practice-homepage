# CSS Practice Homepage
A beginner-friendly project created while following a CSS course. This repository brings together many of the core CSS concepts I learned throughout the course and serves as a personal reference and practice project.

## About
The goal of this project was to practice fundamental CSS concepts by applying them in a single project rather than building isolated exercises. It's not intended to be a production-ready website, but rather a way to reinforce what I've learned and track my progress.

## Topics Covered
This project includes practice with:

    - CSS Fundamentals
    - Typography
    - The Box Model
    - CSS Units
    - Flexbox
    - Responsive Design
    - Backgrounds & Images
    - CSS Animations
    - Positioning
    - CSS Grid
    - CSS Variables (Custom Properties)
    - Pseudo-classes & Pseudo-elements
    - CSS Reset & Normalization
    - Utility Classes 

## Technologies Used

    - HTML5
    - CSS3
